# 14 Model-View-Controller (MVC): Tech Blog

## Description
This is a tech blog where users can make posts and comment on other posts to discuss any topic focused on tech.


## Technologies Used
- Sequelize
- MySQL
- Node.js
- Handlebars.js
- Express.js


## Usage
To run locally... <br />
`npm i ` <br />
`npm run seed` (optional)<br />
`npm run start` <br />
